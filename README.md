# COMP5619_ShiningCoat

### Installing

Open terminal in root directory of the project. Run the following command.

```bash
./run.sh
```

## Running the code

Open terminal in root directory of the project. Run the following command.

```bash
cd backend/
npm run dev
```

Open terminal in root directory of the project. Run the following command.

```bash
cd frontend/
npm start 
```

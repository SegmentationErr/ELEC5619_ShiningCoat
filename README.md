# COMP5619_ShiningCoat

### Installing

Open terminal in root directory of the project. Run the following command.

```bash
./run.sh
```

## Running backend
Open terminal in root directory of the project. Run the following command.

```bash
cd backend/
./gradlew bootRun 
```

## Running frontend
Open terminal in root directory of the project. Run the following command.

```bash
cd frontend/
npm start 
```

## Running Tests
Open terminal in root directory of the project. Run the following command.

```bash
cd backend/
./gradlew test
```
